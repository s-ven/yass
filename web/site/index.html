yass project

mp3 streaming server alowing you to enjoy your audio library from outside your home / flat / room / castle.

current features :
  • navigation "à la" iTunes
  • full flash interface
  • shuffle and loop play
  • track rating
  • fade out at end of track and on track skipping
  • full text search in artist name, album and / or track title
  • users Playlists
  • spectrum Analyzer
planned features :
  • multi users support
  • user configuration (columns, player parameters, fade out durations...)
  • skin customization
  • on the fly re-encoding
  • ogg, aac support
  • web radio support
  • ...
